# Python Employee Management System

## Project Overview

This is a Python-based Employee Management System that allows users to manage employee records using a menu-driven application. The project demonstrates core Python concepts such as functions, loops, conditions, lists, dictionaries, and exception handling.

## Features

* Add Employee
* View All Employees
* Search Employee by ID
* Update Employee Details
* Delete Employee
* Exit Application

## Technologies Used

* Python 3.x
* Visual Studio Code
* Git
* GitHub

## How to Run the Project

1. Clone the repository:

   ```bash
   git clone <repository-link>
   ```

2. Navigate to the project folder:

   ```bash
   cd python-backend-training
   ```

3. Run the program:

   ```bash
   python employee_management.py
   ```

4. Follow the on-screen menu to manage employee records.
